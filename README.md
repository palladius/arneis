# 🦞🍾 Arneis

> "Un buon arnese è come un buon vino: deve essere bilanciato, strutturato e pronto all'azione."

Arneis è un AI Harness sperimentale ispirato a [questo articolo su Claude Code](https://levelup.gitconnected.com/building-claude-code-with-harness-engineering-d2e8c0da85f0) e basato sulla baseline di Fareed Khan.

Il nostro obiettivo è creare un'interfaccia di "Harness Engineering" costruita interamente attorno all'ecosistema **Google Gemini**, con un tocco di orgoglio italiano.

## ♊ Filosofia
- **Gemini First:** Ottimizzato per Gemini 2.5 Flash e Pro.
- **Dual Mode:**
  - ⚡ **Fast Mode:** Gemini 2.5 Flash per risposte istantanee.
  - 🐌 **Slow/Free Mode:** Local Gemma (via Ollama) per task meno urgenti o offline.
